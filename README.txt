Getting Started:
These instructions will get you a copy of the project up and running on your local machine for development and 
testing purposes. See deployment for notes on how to deploy the project on a live system.

Prerequisites:
-Python 3.7.3

Installing:
-Visit the python website and download version 3.7.3 for your prefered operating system.
https://www.python.org/downloads/

-Run the installer and follow the steps.

Running script:
-Once you've installed Python, clone this repository to your local machine.

-Navigate to the the directory in which you've cloned the project.

-Run the script 'csv_writer.py'.

-The new csv filed will override 'new_output.py'.
